# xx-wechat
恋爱时钟微信小程序

恋爱时钟小程序（微信搜胡杨和彩虹）是一款恋爱时钟小程序。无需后台技术，只需要在微信公众平台注册上传代码即可部署使用。

PC 版本参见： https://github.com/chengyumeng/xx
